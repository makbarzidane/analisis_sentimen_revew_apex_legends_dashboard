# Dashboard Apex WordCloud Fix v2

Gunakan file baru ini agar tidak tertukar dengan dashboard lama.

## File yang perlu ada dalam satu folder
- app_dashboard_apex_wordcloud_fix_v2.py
- requirements_dashboard_wordcloud_fix_v2.txt
- apex_sentiment_model_final.pkl
- dataset_final_rejoined.csv

## Cara menjalankan
python -m pip install -r requirements_dashboard_wordcloud_fix_v2.txt
python -m streamlit run app_dashboard_apex_wordcloud_fix_v2.py

Jika masih muncul dashboard lama, berarti terminal masih berada di folder yang salah atau Streamlit lama belum dimatikan.
